# 🛡️ Password Analyzer & Leak Checker

Современное веб-приложение для анализа надежности паролей и проверки их на наличие в базах утечек. 

## ✨ Основные возможности
- **Локальный анализ:** Проверка сложности пароля (энтропия, время взлома) происходит прямо в браузере.
- **k-Anonymity:** Проверка утечек через API *HaveIBeenPwned* без отправки самого пароля на сервер.
- **Генератор паролей:** Создание надежных комбинаций.
- **Безопасность:** Использование заголовков Helmet, Rate-limiting и строгих правил CORS.

## 🌍 Запуск в Production (Чтобы все могли зайти по ссылке)

Если вы хотите развернуть проект на удаленном сервере (VPS) для доступа всех пользователей интернета, вам понадобится облачный сервер (например, Timeweb, Beget, DigitalOcean) с установленным Linux (Ubuntu).

**1. На сервере клонируйте репозиторий:**
```bash
git clone https://github.com/password-analyzer.git
cd password-analyzer
```

**2. Запустите Production-версию:**
В этой версии фронтенд собирается (build) и раздается быстрым веб-сервером **Nginx**. Он же перенаправляет (проксирует) запросы к бэкенду.
```bash
docker-compose -f docker-compose.prod.yml up -d --build
```

**3. Готово!**
Теперь любой пользователь в интернете может ввести IP-адрес вашего сервера (или привязанный домен) в браузере и пользоваться сайтом. 
*Примечание: для продакшена рекомендуется позже привязать домен и подключить SSL (HTTPS) через Certbot.*

## 📖 Как это работает (Безопасность)
Мы используем протокол **k-Anonymity**. При проверке утечек ваш пароль хешируется (SHA-1). На сервер отправляются только первые 5 символов хеша. Сервер возвращает список всех возможных окончаний хешей, которые начинаются с этих 5 символов. Поиск совпадения происходит локально в вашем браузере. Таким образом, ни сервер, ни злоумышленники не могут узнать ваш пароль.
